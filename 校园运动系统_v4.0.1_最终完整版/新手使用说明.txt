校园运动健康数据中台 v4.0.1 使用说明
========================================

一、第一次运行

1. 必须先把 ZIP 压缩包完整解压，不能在压缩包里面直接运行。
2. 解压后双击“安装依赖并启动.bat”。
3. 等待依赖安装完成，登录窗口会自动出现。

二、以后运行

直接双击“直接启动.bat”。

三、默认账号

学生：20250101 / 123456
教师：T001 / teacher123
管理员：A001 / admin888

默认账号第一次登录时必须修改密码。修改成功后才会进入主界面。

四、在 PyCharm 中运行

请直接打开解压后的整个文件夹，确认左侧能同时看到：
main.py、auth_ui.py、requirements.txt、database.py。

然后右键 main.py，选择“运行 main”。

五、常见问题

1. 找不到 requirements.txt：说明打开了上一级目录，请重新打开包含 main.py 的文件夹。
2. ttkbootstrap 红色波浪线：执行 python -m pip install -r requirements.txt。
3. 终端不能输入：程序正在运行，按 Ctrl+C 或点击红色停止按钮。
4. 登录按钮看不到：本修复版已将登录窗口调整为 520×650。
5. 登录后无界面：默认密码需要先修改；本修复版会把改密窗口置顶显示。
